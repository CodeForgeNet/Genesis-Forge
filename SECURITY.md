# Security Policy

## Supported Versions

Specifically focus on the latest stable versions of Genesis Forge.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in Genesis Forge, please **do not** open a public issue. Instead, please report it to us by following these steps:

1. Send an email to [INSERT CONTACT EMAIL] (or use the GitHub "Report a vulnerability" feature if enabled).
2. Include a detailed description of the vulnerability, including steps to reproduce it.
3. We will acknowledge your report within 48 hours and provide a timeline for a fix.

We appreciate your help in keeping Genesis Forge secure!
